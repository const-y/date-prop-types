# date-prop-types

Validators for use with react propTypes

## Installation
```shell
npm install date-prop-types --save-dev
```

## Usage
```javascript
import { date } from 'date-prop-types';

MyComponent.propTypes = {
    hiredDate: date('YYYY-DD-MM')
};
````
